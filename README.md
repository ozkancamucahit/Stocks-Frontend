
# Stocks app 
This software tracks stock information using **financialmodelingprep API**

## Built With
* **React 18** 
* **Tailwind CSS**
* **.NET8 API** 
* **Entity Framework Core** 
* **Identity Framework** 
* **Microsoft Sql Server 2022**
